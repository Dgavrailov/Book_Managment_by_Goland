Documentation:
За да стартирате сървъра отворете терминал в IDE и въведете  "go run main.go". Успешно ще е, ако получите съобщение:
  -- Сървърът е инициализиран успешно на порт 8080...

За да направите заявки с постман:

1. GET - Получаване на всички книги
Въведете URL: http://localhost:8080/books в Postman.
Ще получите отговор с всички книги. Ако искате да ограничите броя на върнатите книги, добавете параметър, например: http://localhost:8080/books?limit=2.
2. GET - Получаване на конкретна книга по ID
Въведете URL: http://localhost:8080/books/<N> (заменете <N> с ID-то на книгата, която искате да получите).
3. POST /books - Създаване на нова книга
Въведете URL: http://localhost:8080/books.
В таба Body изберете raw и JSON.
Въведете следния JSON пример:
{
    "title": "Уни ФМИ",
    "isbn": "1020304050",
    "author": "Михаил Петров",
    "published_at": 2024
}
4. PUT /books/{id}
Въведете URL: http://localhost:8080/books/<N>
в таба Body въведете следния JSON:
{
    "title": "Обновено заглавие",
    "isbn": "0987654321",
    "author": "Петър Петров",
    "published_at": 2024
}
5. DELETE /books/{id}
Въведете URL: http://localhost:8080/books/<N>
Ако книгата е изтрита успешно, ще получите статус код 404 - Page not found.
